## What is the ESA Map Configuration Tool?

This tool allows you to build complex geospatial map configurations without writing code. You can:

- **Create and manage map layers** from various data sources
- **Organize layers** into logical interface groups
- **Configure legends, categories, and colormaps** for data visualization
- **Set up temporal layers** with time controls
- **Validate data quality** with built-in QA tools
- **Export configurations** for the ESA viewer
